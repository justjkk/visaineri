Install
Pre-requisites
  1. Django 1.2.x
  2. Django South 0.7.1
  3. Django Pagination
Installation
  1. Downloading Source from Launchpad:

    You can get the source(which includes this INSTALL file) from launchpad using:

    $ bzr branch lp:visaineri
  2. Create the database
  3. Copy localsettings.py.sample to localsettings.py and edit it as required
  4. Add Parsers module to python path
  5. Create database tables and do migration: 
    $ python manage.py syncdb --migrate
  6. Run your local copy now! 
    $ python manage.py runserver
